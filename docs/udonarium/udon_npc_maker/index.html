使い方の説明
ツールの概要 このツールは、ゲームマスターが多数のNPCを設置、使用することを想定して作られたツールです。
名前、画像を設定してチャットパレットにセリフをコピーアンドペーストして、さくっとNPCコマを作成する。
そういった用途を想定して作られたものです。
各種、入力箇所に入力してから、コマ作成ボタンを押すとZIPファイルが生成されます。
正常にコマが生成できなかった場合、エラーメッセージが表示されます。

注意事項 必須の項目は、名前、サイズ、テーブル座標です。
サイズは小数点以下も設定できます。 (例:1.5 など)
チャットパレット設定は、コマを右クリックして表示されるチャットパレットの内容です。
画像を指定できます。画像なしでもコマは作成できます。
左右反転、上下反転を指定することで、配置されるコマが回転した状態で配置されます。

座標について 最初に読み込みを終えたときの視点を基準に、「X」は左右の軸、「Y」は前後の軸、「Z」は高さの軸にそれぞれ対応しています。

初期値(X:0,Y:0,Z:0)のまま、変更しなくても問題ありません。
