    • やってることの解説
      GithubAPI で 最新リリース版のURLを取得して、圧縮ファイルを取得
      それを解凍して、設定ファイル config.yaml を上書きしないように削除してから展開
      終わったら要らないファイルを削除して終了
      いろいろ雑なのでたぶんもっと綺麗に書けると思います
      (v1.0.0: 2019/06/21 更新)
      現在のバージョンと最新のバージョンを確認する処理を追加
      現在のバージョンが最新ではない場合、バージョンアップするかどうかの確認をするようにした
