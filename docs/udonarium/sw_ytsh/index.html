ゆとシートからキャラコマを生成するやつ
ゆとしコマ
説明 
            ゆとシートのページを開いて、このブックマークレット「ゆとしコマ」をクリックすると
            ユドナ用のコマが生成されます

            情報量が無駄に多いです
            いらない項目はコマから削除して使ってください
            チャパレも同様に無駄が多いです
            こちらも、いらない項目は削除して使ってください

            バフ管理もできるようにしました
            コマに値を設定すると、チャパレの変数としてバフの値が反映されます
            これもいらなかったら削除してください

            ゆとシート側にフェロー登録がある場合、フェロー情報も取得します
            チャパレにはフェロー表の簡易ダイスも登録するようにしています
            これもいらなかったら削除するといいです
